# 🧩 Sudoku Solver in Python

This is a fully functional Sudoku Solver built using Python and the backtracking algorithm. The code is written in a modular format, making it clean, reusable, and easy to understand. This project is perfect for demonstrating recursion, DSA, and Python skills in your portfolio.

---

## 📁 Project Structure

PSEUDOKUSOLVER/
├── main.py # Main script to run the solver
└── Solver/
├── init.py # Declares Solver as a Python package
├── printboard.py # Prints the Sudoku board in 3x3 format
├── validornot.py # Checks if a number can be placed at a given cell
└── emptyornot.py # Finds the next empty cell (0) on the board


---

## ▶️ How to Run

1. Make sure Python 3 is installed.
2. Open terminal or command prompt in the project folder.
3. Run:

```bash
python main.py
You will see the original and solved Sudoku printed in the terminal.
